0 · Обобщение и стратегия

Проектът е функциониращ MVP с добра визуална идентичност, 8 езика и многобройни специализирани space-weather страници. Техническото ниво е юношеско — работи, но носи дългове, които ще се усетят при растеж. Приоритетите в този roadmap следват принципа: (1) първо затваряме известните бъгове, (2) после правим приложението наистина да прави това което маркетингът обещава, (3) след това оптимизираме, (4) и чак накрая добавяме нови фичи.

Ключов принцип: Не добавяй нова фича преди съществуващите да са пълноценно функциониращи. Home обещава "PWA", "Push alerts", "AI Assistant" — нито едно не работи реално. Това е по-важно от нова фича.

Какво е готово (commit fab3aaf)

  • ✅ Счупените /profile и /settings линкове премахнати от Navigation
  • ✅ ISS Nominatim rate-limit bug оправен (dependency array [position] → [])
  • ✅ TTL in-memory кеш + single-flight dedup в noaaApi.ts
  • ✅ Миграция за profile.plan колона създадена (но не е пусната в Supabase)
  • ✅ Security patch: @xmldom/xmldom 0.8.12 → 0.8.13

Какво остава

По-долу са изброени 55 задачи в 6 фази. Те са подредени по зависимост и ROI (return on investment), не по severity. Всяка задача има оценка на effort: S (~1-2 часа), M (~半ден-1 ден), L (2+ дни).

10 · Рискове и mitigation

РискВероятностImpactMitigation
Nominatim ban след ISS bug fix не е пуснат High Critical Deploy fab3aaf веднага. Long-term — proxy през Supabase Edge Function с caching.
NOAA API downtime Medium High Cache layer в ТТаска 3.2 (stale-while-revalidate). Последните валидни данни се показват с "Last updated X min ago" badge.
Stripe webhook пропуснат/дублиран Medium High Idempotent insert (UPSERT по stripe_event_id в stripe_events таблица). Retry automatic от Stripe до 3 дни.
Supabase row-level security bypass Low Critical Никога не ползвай service_role от клиент. Редовно проверявай RLS policies със supabase db lint.
Vite 8 upgrade breaks build Medium Medium Направи в отделен branch. Тествай всеки route manually + npm run build. Rollback план — запази package-lock.json на v5.
Push subscription expires / browser clears High Low Handle 410 Gone response от push service — delete subscription. SW resubscribe on next visit.
Claude API cost overrun (AI Assistant) Low Medium Hard rate limit: 50 msgs/month/Premium user в Edge Function. Spending alert в Anthropic dashboard.
